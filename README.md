# 🎤 Rateform — Рэп Рейтинг

Веб-приложение для оценки рэп-треков по критериям. Музыканты загружают свои треки, а другие пользователи их слушают и оценивают.

## ✨ Основной функционал

### 🔐 Авторизация
- Регистрация по Email/пароль
- Вход через Email/пароль
- Вход через Google
- Профиль пользователя с аватаром

### 🎵 Управление треками
- **Загрузка трэков** — загрузи аудиофайл (MP3/WAV) и обложку (PNG/JPG)
- **Информация о треке** — название, исполнитель, жанр, описание
- **Лента треков** — просмотр всех загруженных треков, отсортированных по дате
- **Детальная страница трэка** — информация, плеер, статистика оценок

### 🎧 Плеер
- Встроенный аудиоплеер прямо на сайте
- Контроль воспроизведения (play/pause)
- Прогресс-бар с возможностью перемотки
- Контроль громкости
- Отображение текущего времени и продолжительности

### ⭐ Система оценок
Каждый пользователь может оценить трэк по 4 критериям (по шкале 1-10):

1. **🎵 Beat** — ритм и бит трэка
2. **🎤 Vocals** — качество вокала
3. **🎛️ Production** — продакшн и звук
4. **🔥 Flow** — флоу и структура

**Функции:**
- Автоматический расчет средней оценки
- Разбивка оценок по критериям
- История оценок пользователя (возможность обновить)
- Статистика: общее количество оценок, средняя оценка

### 👤 Профили музыкантов
- **Страница профиля** — информация о музыканте
- **Статистика**:
  - Количество загруженных треков
  - Всего полученных оценок
  - Средняя оценка по всем трекам
- **Все треки музыканта** — галерея всех его треков
- **Быстрый переход** — кликни на имя автора в карточке или деталях трэка

### 📊 Статистика и аналитика
- Средняя оценка каждого трэка
- Разбивка по критериям оценки
- Общее количество оценок
- Рейтинг музыканта

## 🚀 Начало работы

### Требования
- Современный браузер с поддержкой ES6 модулей
- Firebase проект

### Установка

1. **Клонируй репозиторий**
   ```bash
   git clone <repo-url>
   cd Evaluation-app
   ```

2. **Создай Firebase проект**
   - Перейди на [Firebase Console](https://console.firebase.google.com)
   - Создай новый проект
   - Настрой:
     - **Authentication** — Email/пароль + Google
     - **Firestore Database** — create database
     - **Storage** — для файлов треков

3. **Добавь конфиг Firebase**
   - Скопируй конфиг из Firebase Console
   - Вставь в `js/firebase-config.js`

4. **Открой приложение**
   ```bash
   # Просто открой index.html в браузере
   # или используй локальный сервер:
   python -m http.server 8000
   # Затем откройди http://localhost:8000
   ```

## 📱 Использование

### Загрузка трэка

1. Нажми **"+ Загрузить"** в навигации
2. Заполни форму:
   - Название трека
   - Исполнитель (твое имя)
   - Жанр
   - Описание (опционально)
   - Аудиофайл (MP3/WAV)
   - Обложка (PNG/JPG)
3. Нажми **"Загрузить трек"**
4. Готово! Трек появится в ленте

### Оценка трэка

1. Кликни на трэк в ленте
2. Прослушай на встроенном плеере
3. Используй слайдеры для оценки по 4 критериям
4. Нажми **"Сохранить оценку"**
5. Твоя оценка будет учтена в общем рейтинге

### Просмотр профиля

- **Вариант 1:** Кликни на имя автора в карточке трэка
- **Вариант 2:** Кликни на имя автора в деталях трэка
- **Вариант 3:** Перейди по URL `#profile/{userId}`

## 🔐 Безопасность

- Все данные хранятся в защищенной Firestore базе
- Firebase Rules ограничивают доступ:
  - Только авторизованные пользователи могут загружать треки
  - Только авторизованные пользователи могут оценивать
  - Можно редактировать только свои данные
- Аудиофайлы хранятся в Firebase Storage с публичным доступом (чтение)

## 💾 База данных (Firestore)

### Коллекция `users`
```javascript
{
  uid: string,
  username: string,
  email: string,
  avatar: string (URL) | null,
  createdAt: timestamp,
  uploadsCount: number,
  ratingsCount: number
}
```

### Коллекция `tracks`
```javascript
{
  title: string,
  artist: string,
  genre: string,
  description: string,
  uploadedBy: string (uid),
  uploadedByName: string,
  audioUrl: string (Storage URL),
  coverUrl: string (Storage URL),
  averageRating: number (0-10),
  totalRatings: number,
  ratingBreakdown: {
    beat: number,
    vocals: number,
    production: number,
    flow: number
  },
  duration: number (seconds),
  createdAt: timestamp
}
```

### Коллекция `ratings`
```javascript
{
  trackId: string,
  userId: string (uid),
  beat: number (1-10),
  vocals: number (1-10),
  production: number (1-10),
  flow: number (1-10),
  overallRating: number (average),
  createdAt: timestamp,
  updatedAt: timestamp
}
```

## 🔄 Маршруты приложения

- `#feed` — лента всех треков
- `#track/{trackId}` — детальная страница трэка с плеером
- `#upload` — загрузка нового трэка
- `#profile/{userId}` — профиль музыканта
- `#top` — топ треков по рейтингу (опционально)
